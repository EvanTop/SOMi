## SOMi域名展示和管理网站

### SOMI.im v2（[v2版本下载](https://github.com/EvanTop/SOMi/releases/tag/SOMi_v2)） 

- v2在保持前端极致体验的同时，引入了轻量级的 PHP 后端支持。架构由 “纯静态” 升级为 “前后端分离（JSON 存储）”，既保留了无需 MySQL 数据库的轻便特性，又实现了数据的服务端持久化与跨设备同步。

#### 这种架构设计带来了三大核心优势：
- 部署轻量:无需 MySQL 数据库，仅需 PHP 7.4 或 8.0+ 环境支持，资源占用依然极低。
- 响应极快:前端资源经 Vite 编译，数据交互采用本地 JSON 读写，速度远超传统 CMS。
- 多端同步:数据存储在服务器端 JSON 文件中，支持不同电脑/浏览器间的数据无缝同步。
